# QuizAgent - AI-платформа для тестирования

Платформа для тестирования на основе ИИ с поддержкой нескольких языков, админ-панелью и интеграцией двух AI-провайдеров (Gemini и Groq).

## 🚀 Возможности

- **Генерация квизов с помощью ИИ**: Создавайте уникальные тесты на любую тему с использованием Gemini 2.5 Flash или Groq Llama 3.3
- **Поддержка нескольких языков**: Английский, Русский, Казахский
- **Адаптивная сложность**: Уровни Легко, Средне, Сложно
- **Оценка в реальном времени**: Мгновенная обратная связь от ИИ
- **Управление пользователями**: Роли Студент и Администратор
- **История тестов**: Отслеживание всех попыток с детальной статистикой
- **Админ-панель**: Просмотр всех пользователей и результатов тестов
- **Интеграция базы знаний**: ИИ использует кураторскую базу вопросов

### Backend

- **Gemini AI** и **Groq AI**

## 📚 API Endpoints

### Аутентификация
- `POST /auth/register` - Регистрация нового пользователя
- `POST /auth/login` - Вход пользователя
- `GET /auth/me` - Получить текущего пользователя
- `GET /auth/users` - Список всех пользователей (только админ)

### Квизы
- `POST /exam/quiz/generate` - Генерация квиза
- `GET /exam/attempts` - Получить попытки тестов
- `POST /exam/attempts` - Сохранить попытку теста

## 🎯 Использование

### Процесс для студента
1. Регистрация/Вход
2. Выбор темы, сложности, AI модели
3. Ответы на вопросы
4. Просмотр результатов и обратной связи
5. Проверка истории в профиле

### Процесс для администратора
1. Вход с учетной записью администратора
2. Доступ к админ-панели
3. Просмотр всех пользователей
4. Мониторинг всех попыток тестов
5. Отслеживание прогресса студентов

## 🌐 Локализация

Переключайте язык с помощью иконки глобуса в шапке:
- 🇬🇧 English
- 🇷🇺 Русский
- 🇰🇿 Қазақша

Все элементы интерфейса, включая админ-панель, полностью переведены.

## 🤖 AI Провайдеры

### Gemini 2.5 Flash
- Быстрое время отклика
- Высокая точность
- Провайдер по умолчанию

### Groq Llama 3.3 70B
- Сверхбыстрый вывод
- Альтернативный вариант
- Требуется API ключ Groq

Обе AI модели используют одну и ту же базу знаний из директории `ai-knowledge-base/`.

# Overclock Hackathon Project
ФУНКЦИОНАЛ:
- Генерация теста на основе 3 переменных:
      - Предмет теста,
      - Кол-во вопросов,
      - Сложность теста: легкий, средний, сложный
  ( Тест должен сохранять кол-во правильных, неправильных ответов )
  ( в случае количества неправильных вопросов, если больше 50% - кнопочка на подкрепление знаний
      генерация теории + задачи с объяснением )
